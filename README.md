# 游享购电子商城管理系统 python2

## 项目概述


## 技术栈

- 后端：Python Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

- 用户管理
- 商品管理
- 订单管理
- 促销活动
- 数据报表

## 系统角色

- 系统管理员
- 商家
- 普通用户

## 运行环境

- Python 3.8+
- Django 3.x
- Node.js 12.18+
- Vue.js 2.x
- MySQL 5.7+

## 部署步骤

详细的部署步骤请参考随项目提供的部署文档。

## 目录结构

```
project/
├── backend/             # 后端代码目录
├── frontend/            # 前端代码目录
└── docs/                # 项目文档目录
```

## 核心亮点

- 双端分离，便于管理和扩展
- 基于Django ORM，易于数据库操作
- 提供丰富的功能模块，满足不同角色需求
